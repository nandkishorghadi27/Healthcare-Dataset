# Healthcare Data Analytics Project
## Project Objective
This project focuses on analyzing a Healthcare Dataset containing 55,500 patient records to uncover insights related to medical conditions, hospital admissions, billing patterns, and patient demographics.<br>

2. Which admission type occurs most frequently?<br>

## Project Insight
This project analyzes healthcare patient data to identify trends in medical conditions, admissions, and billing patterns. Data was cleaned using Excel, analyzed with PostgreSQL (pgAdmin), and visualized through an interactive Power BI dashboard. The analysis helps transform raw healthcare data into meaningful insights for better decision-making.

## Final Conclusion
This project demonstrates how healthcare data can be transformed into meaningful insights using Excel, PostgreSQL, and Power BI. The analysis helped identify patient trends, common medical conditions, and billing patterns. Overall, it showcases an end-to-end data analytics workflow that supports better healthcare decision-making.
